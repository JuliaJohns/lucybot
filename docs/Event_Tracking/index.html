Event tracking is a great way to understand user behavior and measure success. LucyBot integrates directly with Mixpanel, which has a sizeable free tier. However, you can also use Google Analytics or other event tracking systems.

Mixpanel

Other trackers

You can add the file javascript/tracking.js, and define a function which sends tracking events to the system of your choice. The function should take in an event name, and an object with key-value properties.

For example, to use Google Analytics:

/* Paste Google Analytics setup here */

window.lucybot.tracker = function(name, properties) {
  for (var key in properties) {
    ga('send', {
      hitType: 'event',
      eventCategory: name,
      eventAction: key,
      eventLabel: key,
      eventValue: properties[key],
    })
  }
}
